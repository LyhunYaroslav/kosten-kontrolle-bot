# KostenKontrolleBot 🤖💰

Ein einfacher Telegram-Bot zur Verwaltung persönlicher Ausgaben.

## 📌 Beschreibung

Der FinanzTrackerBot hilft dir dabei, deine täglichen Ausgaben zu speichern, anzuzeigen und zu löschen.  
Alle Daten werden lokal in einer SQLite-Datenbank gespeichert.

---

## 🚀 Funktionen

- ➕ Ausgabe hinzufügen
- 📋 Alle Ausgaben anzeigen
- 🗑 Ausgabe löschen
- 💾 Lokale Speicherung (SQLite)

---

## 🛠 Tech Stack

- Python 🐍
- aiogram (Telegram Bot API)
- SQLite (Datenbank)
- aiosqlite (async Datenbankzugriff)
- python-dotenv (Umgebungsvariablen)
